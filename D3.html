Uvod u d3.js



D3.js je JavaScript biblioteka koja se koristi za rukovanje dokumentima na osnovu podataka. Omogućava bilo koju vrstu vizualizacije podataka. Na ovoj stranici ćemo pokušati da vas što bolje upoznamo sa D3.js.



Naziv D3 potiče od tri slova D (Data-Driven Documents) - Dokumenti vođeni podacima. Ova biblioteka koja se koristi za vizualizaciju JavaScript podataka postala je popularna zbog nekoliko funkcija koje menjaju dosadne i monotone podatke u interaktivne i zabavne:

  • d3 je revolucionaran način vizualizacije podataka stvorene za veb povezivanjem podataka sa elementima na objektnom modelu.
  • d3 je moderna biblioteka koja nudi širok spektar unapred definisanih primera
  • d3 ostavlja mnogo prostora za kreativno prilagođavanje, jer se elementi na koje su podaci vezani mogu stilizovati kao bilo koji drugi HTML elementi.
  • d3 podržava interakciju koju pokreću događaji u pregledaču

Šta je HTML?


  • HTML je jezik za označavanje hiperteksta, jezik specijalno namenjen opisu veb stranica. Pomoću njega se jednostavno mogu odvojiti elementi kao što su naslovi, paragrafi, citati i slično. Veb pregledači poput Mozile ili Google Chrome-a čitaju ovu vrstu datoteke i prevode na veb stranicu.
  • U HTML fajlovima pišemo elemente koji sačinjavaju veb stranicu. Na primer, naslov prvog nivoa predstavlja h1 tag, paragraf p tag, slika se označava sa img tagom i još mnogi drugi.
  • Bez osnovnog znanja HTML-a nije moguće napraviti bilo šta u d3.js biblioteci. Ostanite sa nama do kraja!

→ Objašnjenje:
  • CSS (Cascading Style Sheet) je jezik koji koristimo za oblikovanje i stilizovanje HTML dokumenata. Omogućava primenu specifičnih stilova.
  • Ne moguće je napraviti d3.js dokument bez osnovnog znanja o css-u. Od pomoći bi mogao da vam bude ovaj tutorijal.


Napravite oblike pomoću SVG

→ Objašnjenje:
  • SVG - označava skabilnu vektorsku grafiku. To je format vektorske slike. U osnovi to je jezik koji omogućava stvaranje oblika pomoću koda.
  • Šta je SVG?
    • SVG označava skabilnu vektorsku grafiku
    • SVG se koristi za definisanje vektorske grafike na Vebu
    • Svaki element i svaki atribut u SVG fajlovima se može animirati
    • SVG slike se mogu pretraživati, indeksirati i skriptovati
    • SVG grafika ne gubi kvalitet ako je zumirana ili promenjena veličina
  • D3.js grafikon je zapravo skup SVG oblika.

Izmenite elemente pomoću JavaScript i D3.js

→ Objašnjenje:
  • JavaScript je jedan od tri osnovne tehnologije WWW-a. Omogućava interakciju našeg Veb sajta.
  • D3.js je JavaScript biblioteka posebno korisna za vizualizaciju podataka. Omogućava kreiranje, odabir i izmenu elemenata.

Koordinatni sistem

→ Objašnjenje:
  • Izgradnja D3.js grafikona počinje stvaranjem SVG elemenata.
  • Bitno je napomenuti da:
    • gornji levi ugao ima koordinate x=0 i y=0;
    • donji levi ugao ima koordinate x=0 i y=height;
    • gornji desni ugao ima koordinate x=width i y=0.

→ primer: 
<!DOCTYPE html>
  
  <!-- Dodavanje SVG-a -->
  <svg id="dataviz_area" height=200 width=450></svg>
  
  <!-- Učitavanje d3.js -->
  <script src="https://d3js.org/d3.v4.js"></script>
  
  
<script>
            var svg = d3.select("#dataviz_area")
            svg.append("circle")
              .attr("cx", 2).attr("cy", 2).attr("r", 40).style("fill", "blue");
            svg.append("circle")
              .attr("cx", 140).attr("cy", 70).attr("r", 40).style("fill", "red");
            svg.append("circle")
              .attr("cx", 300).attr("cy", 100).attr("r", 40).style("fill", "green");
          </script>

Od podataka do piksela: Scale

→ Objašnjenje:
  • Pozicija elemenata je podešena da bude u pikselima.
  • Zbog toga nam je potrebna funkcija koja numeričku promenljivu prevodi u piksele na određenoj poziciji. Ta funkcija se zove scale
  • Ako imamo podatke u procentima i određena SVG površina je širine 400px. 0% -> 0 px. 100% -> 400 px. 50% -> 200px.
  • Ako izvršite liniju koda x(10), d3 će vratiti vrednost u pikselima.

<!DOCTYPE html>
      
      <!-- Dodavanje SVG-a -->
      <svg id="viz_area" height=200 width=450></svg>
      
      <!-- Učitavanje d3.js -->
      <script src="https://d3js.org/d3.v4.js"></script>
      
      
<script>
            // Select the svg area
            var svg = d3.select("#viz_area")

            // Create a scale: transform value in pixel
            var x = d3.scaleLinear()
              .domain([0, 100])         // This is the min and the max of the data: 0 to 100 if percentages
              .range([0, 400]);       // This is the corresponding value I want in Pixel
            // Try console.log( x(25) ) to see what this x function does.

            // Add 3 dots for 0, 50 and 100%
            svg.append("circle")
              .attr("cx", x(10)).attr("cy", 100).attr("r", 40).style("fill", "blue");
            svg.append("circle")
              .attr("cx", x(50)).attr("cy", 100).attr("r", 40).style("fill", "red");
            svg.append("circle")
              .attr("cx", x(100)).attr("cy", 100).attr("r", 40).style("fill", "green");
          </script>

Dodavanje ose

→ Objašnjenje:
  • D3 nudi nekoliko funkcija za automatsko crtanje ose
  • Ove ose su uvek nacrtane na vrhu scale. Ova skala određuje gde se osa mora postaviti i koji opseg treba da obuhvati.
  • Funkcija axisBottom() pravi horizontalnu osu, sa svim oznakama. axisLeft() se koristi za Y osu.

Margin & translation

→ Objašnjenje:
  • Položaj ose često treba prilagoditi. Na primer, X osa se obično postavlja pri samom dnu grafikona.
  • To je omogućeno zahvaljujući "translation-u". U osnovi, primena .attr("transform", "translate(20,50)") - 20px desno i 50px - pri dnu.
  • Vrlo česta strategija je da se translation primeni u SVG oblasti, stvarajući margine oko grafikona. Ovo je zaista važno razumeti jer skoro svi d3.js grafikoni tako započinju.

<!DOCTYPE html>
      
      <!-- Dodavanje SVG -->
      <div id="Area"></div>
      
      <!-- Učitavanje d3.js -->
      <script src="https://d3js.org/d3.v4.js"></script>
      
      
<script>

            // set the dimensions and margins of the graph
            var margin = { top: 10, right: 40, bottom: 30, left: 30 },
              width = 450 - margin.left - margin.right,
              height = 400 - margin.top - margin.bottom;

            // append the svg object to the body of the page
            var sVg = d3.select("#Area")
              .append("svg")
              .attr("width", width + margin.left + margin.right)
              .attr("height", height + margin.top + margin.bottom)
              // translate this svg element to leave some margin.
              .append("g")
              .attr("transform", "translate(" + margin.left + "," + margin.top + ")");

            // X scale and Axis
            var x = d3.scaleLinear()
              .domain([0, 100])         // This is the min and the max of the data: 0 to 100 if percentages
              .range([0, width]);       // This is the corresponding value I want in Pixel
            sVg
              .append('g')
              .attr("transform", "translate(0," + height + ")")
              .call(d3.axisBottom(x));

            // X scale and Axis
            var y = d3.scaleLinear()
              .domain([0, 100])         // This is the min and the max of the data: 0 to 100 if percentages
              .range([height, 0]);       // This is the corresponding value I want in Pixel
            sVg
              .append('g')
              .call(d3.axisLeft(y));

          </script>

Vezivanje podataka

→ Objašnjenje:
  • Vezivanje podataka za SVG elemente je poslednji korak koji nam je potreban da bismo razumeli sve ovo dosadašnje. Ovo je možda i najteži deo za razumevanje.
  • Možete pratiti sledeće korake:
    • SVG: oblast gde se grafikon nalazi
    • .selectAll("whatever"): izaberite sve elemente koji još nisu napravljeni
    • .data(data): navedite podatke koje ćete koristiti
    • .enter(): pokreće petlju za podatke
    • .append("circle"): za svaku iterakciju dodajte krug
    • .attr("cx", function(d){ return x(d.x) }): daje X položaj kruga. d će biti data[0]. x(d.x) je odgovarajuća pozicija u pikselima pronađena zahvaljujući x scale.

→ Primer: 
<!DOCTYPE html>
  
  <!-- Dodavanje SVG -->
  <div id="scatter_area"></div>
  
  <!-- Učitavanje d3.js -->
  <script src="https://d3js.org/d3.v4.js"></script>
  
  
<script>
  
  // set the dimensions and margins of the graph
  var margin = {top: 10, right: 40, bottom: 30, left: 30},
      width = 450 - margin.left - margin.right,
      height = 400 - margin.top - margin.bottom;
  
  // append the svg object to the body of the page
  var svG = d3.select("#scatter_area")
    .append("svg")
      .attr("width", width + margin.left + margin.right)
      .attr("height", height + margin.top + margin.bottom)
    .append("g")
      .attr("transform",
            "translate(" + margin.left + "," + margin.top + ")");
  
  // Create data
  var data = [ {x:10, y:20}, {x:40, y:90}, {x:80, y:50} ]
  
  // X scale and Axis
  var x = d3.scaleLinear()
      .domain([0, 100])         // This is the min and the max of the data: 0 to 100 if percentages
      .range([0, width]);       // This is the corresponding value I want in Pixel
  svG
    .append('g')
    .attr("transform", "translate(0," + height + ")")
    .call(d3.axisBottom(x));
  
  // X scale and Axis
  var y = d3.scaleLinear()
      .domain([0, 100])         // This is the min and the max of the data: 0 to 100 if percentages
      .range([height, 0]);       // This is the corresponding value I want in Pixel
  svG
    .append('g')
    .call(d3.axisLeft(y));
  
  // Add 3 dots for 0, 50 and 100%
  svG
    .selectAll("whatever")
    .data(data)
    .enter()
    .append("circle")
      .attr("cx", function(d){ return x(d.x) })
      .attr("cy", function(d){ return y(d.y) })
      .attr("r", 7)
  
  
  </script>
